# Analytics tracker guide

Use `ArticlePerformanceTracker.csv` for portability or the formatted workbook for ongoing reviews. Enter one row per article per reporting month so trends remain auditable.

## Metric definitions

| Metric | Definition |
|---|---|
| Organic impressions | Search-result appearances during the reporting month |
| Organic clicks | Search-result clicks during the reporting month |
| Organic CTR | Organic clicks divided by impressions |
| Average position | Search Console average position for the page/query scope used consistently |
| Organic sessions | Sessions attributed to organic search |
| Engaged sessions | Sessions meeting the configured analytics engagement definition |
| Engagement rate | Engaged sessions divided by organic sessions |
| Affiliate clicks | Outbound tracked clicks by retailer |
| Affiliate CTR | Total affiliate clicks divided by organic sessions |
| Conversions | Confirmed attributable purchases or program conversions when available |
| Estimated commission | Estimated, not recognized, commission for the reporting month |

## Monthly process

1. Use calendar months and one timezone consistently.
2. Export page-level Search Console and analytics data using the same filters each month.
3. Record retailer clicks from the approved tracking source; do not mix raw page clicks with program-reported conversions.
4. Add notes for launches, internal-link changes, title tests, seasonality, outages, promotions, or affiliate-link changes.
5. Never overwrite a prior month's row. Corrections should be documented in Notes.

## Review signals

- High impressions and low CTR: review title, description, intent match, and rich-result eligibility.
- Healthy clicks and weak engagement: improve the opening, page speed, scannability, or intent alignment.
- Strong project engagement and weak continuation: strengthen next-project and tutorial links.
- Strong sessions and weak affiliate CTR: confirm product relevance, disclosure placement, and destination availability without adding pressure.
- Clicks without reported conversions: verify tracking, retailer market, product availability, and attribution windows.

Treat small samples cautiously. Compare at least three periods and annotate material changes before drawing conclusions.
