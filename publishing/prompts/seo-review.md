# Prompt: SEO Review

Review the specified page for useful, accurate search presentation without keyword stuffing.

Verify one clear H1, a distinct SEO title, a specific meta description, Open Graph metadata, canonical URL, descriptive image alt text, semantic headings, structured data, helpful internal links, and alignment between search copy and page content. Confirm that the title and description are unique in the generated site. Do not add claims, urgency, or repetitive keywords merely for ranking.

Run the production build and build audit. Report recommended or completed changes with validation evidence.
