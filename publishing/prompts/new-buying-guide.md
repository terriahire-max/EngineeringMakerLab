# Prompt: New Buying Guide

Create one original Engineering Maker Lab buying guide using `publishing/templates/buying-guide.md` and products already present in the centralized catalog.

Identify the reader and decision, explain transparent evaluation criteria, distinguish real tradeoffs, and recommend only suitable products. Never invent prices, availability, quantities, reviews, or product features. Avoid hype, fake urgency, and affiliate-first language. Keep retailer URLs in the centralized affiliate configuration and include the affiliate disclosure.

Verify product IDs, internal links, metadata, accessibility, and mobile readability. Run all configured checks and the production build, then report validation results and any product details needing verification.
