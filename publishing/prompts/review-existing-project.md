# Prompt: Review Existing Project

Review one existing Engineering Maker Lab project against the current schema, reusable components, and all checklists in `publishing/checklists/`.

Check factual and electrical consistency, original writing, beginner clarity, GPIO assignments, code behavior, parts, official resources, affiliate buttons, visuals, accessibility, SEO metadata, internal links, mobile presentation, troubleshooting, FAQ, and next-project behavior. Do not redesign the site or create another project.

Correct only in-scope issues. Run Astro/content checks, the production build, and the build audit. Report evidence, files changed, remaining external dependencies, and validation results.
