# Prompt: New Project

Create one original Engineering Maker Lab project using `publishing/templates/project.md` without changing the website layout or navigation.

Inputs: title, slug, platform, difficulty, time, cost, recommended product ID, learning goals, parts, GPIO assignments, official resource URLs, related content, and planned next project.

Requirements:

- Write for a complete beginner using short paragraphs and defined technical terms.
- Keep every pin number consistent across prose, diagrams, and tested code.
- Link to official vendor resources; never reproduce or paraphrase a vendor tutorial.
- Use the centralized product catalog and affiliate configuration. Never hardcode retailer URLs.
- Create original, accessible visuals with fixed dimensions and descriptive alt text.
- Include practical troubleshooting, original FAQs, and an honest recommendation without hype.
- Verify that every internal route exists. Mark unpublished follow-ups as planned without linking them.
- Run the repository content check, build, and build audit before reporting completion.

Output a concise change summary, validation results, and any missing verified URLs, photos, or product facts.
