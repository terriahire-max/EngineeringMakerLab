# Prompt: New Tutorial

Create one original Engineering Maker Lab tutorial using `publishing/templates/tutorial.md` and the existing tutorial architecture.

Explain the concept from first principles, define unfamiliar terms, use short paragraphs, and connect the lesson to a practical electronics task. Include one realistic example, common beginner mistakes, relevant safety boundaries, and only verified internal links. Do not copy vendor documentation. Use an original image when a visual materially improves understanding, with descriptive alt text and fixed dimensions.

Run all configured content checks and the production build. Report created files, validation results, and any facts or assets still requiring verification.
