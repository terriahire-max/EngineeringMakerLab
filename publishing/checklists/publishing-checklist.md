# Publishing Checklist

## Before Publish

- [ ] Project, Tutorial, or Buying Guide checklist is complete.
- [ ] Original writing and source attribution are confirmed.
- [ ] Official resources and internal links are verified.
- [ ] Affiliate buttons and disclosure are verified.
- [ ] Images, circuit diagram, and breadboard diagram are complete when required.
- [ ] Mobile and accessibility review is complete.
- [ ] Astro/content checks pass.
- [ ] Production build passes.
- [ ] Build audit passes with no errors.

## Version Control and Deployment

- [ ] Review the diff and exclude unrelated or generated files.
- [ ] Commit with a specific message describing the published content.
- [ ] Push the intended branch only after approval.
- [ ] Confirm Cloudflare deployment succeeds.
- [ ] Smoke-test the live canonical URL, images, links, and purchase buttons.
- [ ] Record any follow-up work without presenting unfinished items as published.
