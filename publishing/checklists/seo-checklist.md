# SEO Checklist

- [ ] SEO title is accurate, specific, and unique.
- [ ] Meta description is useful, specific, and unique.
- [ ] One descriptive H1 matches the page purpose.
- [ ] Heading order is semantic.
- [ ] Canonical URL matches the generated route.
- [ ] Open Graph title, description, image, alt text, and URL are present.
- [ ] Structured data matches visible content.
- [ ] Image alt text describes useful visual information.
- [ ] Internal links are relevant and valid.
- [ ] Keywords appear naturally without repetition or stuffing.
- [ ] Search copy contains no unsupported promises or claims.
- [ ] Generated-site audit reports no duplicate metadata or broken links.
